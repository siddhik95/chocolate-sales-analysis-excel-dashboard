# Chocolate Sales Analysis (Excel)

## Overview
- Analysed 500+ rows of sales data using Advanced Excel
- Built interactive dashboard using Pivot Tables, Charts, and Slicers
- Tracked sales by region, category, and channel

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Key Insights
- Central region is the top-performing region
- Dark chocolate contributes ~20% of total sales
- Identified monthly sales trends and peak periods
